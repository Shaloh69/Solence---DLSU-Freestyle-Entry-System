# textures/materials/

CC0 PBR texture sets, organized by material family subfolder
(`wood/ fabric/ metal/ concrete/ glass/` — create as assets land).

Conventions:

- **Sources:** Poly Haven (primary), ambientCG (supplementary). Both
  CC0; still verify per-download.
- **Resolution policy:** store 1K in the repo (Showcase Mode is
  stylized low-poly — 1K reads fine); link the source page in the
  license log so 2K–8K can be fetched if ever needed. Don't commit
  multi-K sets that bloat the repo.
- **Naming:** `<family>/<name>_<map>.<ext>` with standard map suffixes:
  `albedo`, `normal`, `roughness`, `metallic`, `height`, `ao`
  (e.g. `wood/oak-floor_albedo.jpg`).
