# models/windows/

Same rule as `../doors/`: **windows are PARAMETRIC/GENERATED, not
static meshes** (Phase 2 brief §4.1). In-product windows are generated
at the opening's exact width and the sill/head band heights by
`client/lib/opening-geometry.ts`; this folder holds reference
meshes/profiles only, for studying proportions when authoring a new
window family. Nothing here is loaded at runtime.

CC0 only; verify per-asset. See `../../README.md` for the policy.
